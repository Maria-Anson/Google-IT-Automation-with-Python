> When using regular expressions, which of the following expressions uses a reserved character that can represent any single character? 
> 
>  re.findall(f*n, text) 
> 
>  re.findall(^un, text) 
> 

      re.findall(f.n, text) 
> 
>  re.findall(fu$, text) 
> 
> Check
> 
> Correct
> 
> Nailed it! The dot (.) represents any single character.
